# Polo Club Newsletter

This newsletter template is based on the St. Andrews Country Club newsletter design.

## What You Need to Provide

To complete this newsletter, please provide the following information:

### 1. Header Information
- Newsletter date (currently set to "January 2026")

### 2. Agent Photo & Information
- Agent photo URL (replace `YOUR_PHOTO_URL_HERE`)
- Agent name (replace `YOUR AGENT NAME`)
- Brokerage name (replace `YOUR BROKERAGE NAME`)
- Phone number (replace `YOUR PHONE NUMBER`)

### 3. Market Statistics (Summary Stats section)
- Price Range (e.g., "$1.995M - $10.5M")
- Number of Active Listings
- Number Sold Year-to-Date
- Absorption Rate percentage

### 4. Featured Listings
For each listing, provide:
- Address
- Price
- Number of bedrooms
- Number of bathrooms
- Living square footage
- Year built
- Lot square footage
- Days on market
- MLS number
- Listing URL
- Feature tags (e.g., "Waterfront", "New Construction", "Price Reduced")

You can duplicate the listing template block for multiple properties.

### 5. Call-to-Action URLs
- Contact page URL (replace `YOUR_CONTACT_URL_HERE`)
- View all listings URL (replace `YOUR_WEBSITE_URL_HERE`)

### 6. Market Insights
- Custom market commentary about Polo Club

## Color Scheme

The newsletter uses:

- Red (New Construction): `#dc3545`
